# Unveiling the Android App Market: Google Play Store Data Analysis
![Unveiling App](https://github.com/AgasthiDoshi/OIBSIP_DataAnalytics_Task8/blob/main/20251226_1411_Data%20Analytics%20Presentation%20Cover_simple_compose_01kdcx261he2zv90vfz8x0rvmk.png)
## Project Objective
The objective of this project is to analyze Google Play Store
application data to understand app market dynamics. The analysis
focuses on cleaning the data, exploring app categories, and
examining key metrics such as ratings, installs, and pricing trends.

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
- Loaded and explored the Google Play Store dataset
- Cleaned and corrected data types for accurate analysis
- Analyzed app distribution across categories
- Examined ratings, installs, and pricing patterns
- Visualized key insights using plots and charts

## Outcome & Learnings
- Most apps on the Play Store are free
- Certain categories dominate the app market
- Higher-rated apps generally receive more installs
- Data visualization helps uncover market trends effectively

## Project Files
- `Task8_Google_Play_Store_Analysis.ipynb` – Main analysis notebook
- `apps.csv` – Dataset used for analysis
